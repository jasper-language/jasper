# 第九章：表达式

## 9.1 表达式入口

```antlr
expression
    : lambdaExpression
    | assignmentExpression
    ;
```

## 9.2 Lambda

Lambda 使用 `lambda (...) { ... }`：

```jasper
var zero = lambda () {
    return 0;
};

var add = lambda (a: int32, b: int32) {
    return a + b;
};
```

不支持无括号单参数 lambda，也不使用 `=>` 作为正式语法。
Lambda 不能带 identifier，也不在 lambda 语法里声明返回类型；返回类型由左侧目标类型或 checker 推断。

## 9.3 赋值

```jasper
x = 42;
count += 1;
person.name = "Ada";
```

## 9.4 Null 表达式

`??` 是表达式层 null coalescing：

```jasper
var name: string = maybeName ?? "anonymous";
```

类型层不存在 `T??`。

方法风格可由标准库提供：

```jasper
var name = maybeName.or("anonymous");
var user = maybeUser.expect("missing user");
```

## 9.5 一元表达式

```antlr
unaryExpression
    : '+' unaryExpression
    | '-' unaryExpression
    | '*' unaryExpression
    | '&' unaryExpression
    | '~' unaryExpression
    | (Not | '!') unaryExpression
    | awaitExpression
    | goExpression
    | castExpression
    | primary
    ;
```

指针相关：

```jasper
var x: int32 = 1;
var p: *int32 = &x;
var y: int32 = *p;
```

- `&expr` 是取地址表达式。
- `*expr` 是解引用表达式。
- `&T` 不是类型语法。

## 9.6 Go 和 Await

`go` 启动协程语法糖，`await` 等待 Future：

```jasper
go task();
var result = await future;

go lambda () {
    workerLoop();
};
```

## 9.7 Cast

当前保留 cast 表达式入口：

```jasper
var n = (int32) value;
var user = cast value as User;
```

交集类型 cast 如 `(A & B) value` 不作为第一版核心类型设计。

## 9.8 Primary 和后缀

常见 primary：

```jasper
42
"hello"
true
this
super
new User()
(a + b)
```

常见后缀：

```jasper
fn(args)
obj.field
obj.method()
arr[i]
maybe?.field
```
