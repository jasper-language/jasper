# 第五章：函数

## 5.1 函数声明

Jasper 使用 `function` 声明函数，返回类型放在参数列表之后：

```jasper
function add(a: int32, b: int32): int32 {
    return a + b;
}

function log(msg: string): void {
    println(msg);
}
```

省略返回类型等价于 `void`：

```jasper
function noop() { }
```

## 5.2 参数

参数使用统一绑定语法 `name: Type`：

```jasper
function move(x: int32, y: int32): void {
}
```

参数类型可由后续设计决定是否允许省略；当前推荐显式写出。

## 5.3 多值返回

多值返回使用 tuple 类型：

```jasper
function divide(a: int32, b: int32): (int32, int32) {
    return a / b, a % b;
}

var q, r = divide(17, 5);
```

`return a, b` 按 Go 风格视为多返回；`var a, b = expr` 按位置绑定多返回值，数量必须匹配。

## 5.4 方法和构造器

类内函数仍使用 `function`：

```jasper
class Counter {
    private var value: int32 = 0;

    public constructor(initial: int32) {
        this.value = initial;
    }

    public function inc(): void {
        this.value = this.value + 1;
    }
}
```

## 5.5 Lambda

Lambda 使用 `lambda (...) { ... }`。

```jasper
var zero = lambda () {
    return 0;
};

var add = lambda (a: int32, b: int32) {
    return a + b;
};

var sum = lambda (a, b) {
    return a + b;
};
```

Lambda 参数必须使用括号；不支持单参数无括号 lambda。

对应 parser：

```antlr
lambdaExpression
    : Lambda lambdaParameters block
    ;

lambdaParameters
    : '(' ( Identifier (',' Identifier)* ','? )? ')'
    | '(' plist=functionParameterList? ')'
    ;

```

Lambda 是匿名表达式，不能带 identifier，也不在 lambda 语法里声明返回类型。返回类型由左侧目标类型或 checker 推断。

```jasper
var f: *T = lambda () {
    return value;
};
```

## 5.6 泛型函数

```jasper
function first<T>(items: T[]): T? {
    return items[0];
}

function max<T extends Comparable<T>>(a: T, b: T): T {
    return a;
}
```

`where` 可用于额外约束：

```jasper
function sort<T>(items: T[]): T[]
where T is Comparable {
    return items;
}
```

## 5.7 异常声明

Jasper 当前确认异常控制流使用：

```jasper
throw expr;
try { } catch (e: Error) { } finally { }
```

`throws` 保留为函数签名元信息，不做 checked exception。它可供 IDE、文档生成、静态分析使用，但不强制调用方 `catch`。

```jasper
function read(path: string): string throws IOException {
    return "";
}
```

实际异常控制流仍然只由 `throw + try/catch/finally` 表达。
