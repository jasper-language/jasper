# 第八章：Switch

Jasper 没有 `match` 语法。当前只确认 `switch / case / default`。

## 8.1 Switch 语法

```antlr
switchStatement
    : Switch '(' expression ')' '{' switchCaseClause* defaultClause '}'
    ;

switchCaseClause
    : Case expression FAT_ARROW block
    ;

defaultClause
    : Default FAT_ARROW block
    ;
```

示例：

```jasper
switch (code) {
    case 200 => {
        println("OK");
    }
    case 404 => {
        println("missing");
    }
    default => {
        println("unknown");
    }
}
```

## 8.2 设计规则

- `switch` 必须包含一个 `default`。
- `default` 只能出现一次。
- `default` 必须放在最后。
- `case` 使用 `=> block`，不支持 fallthrough。
- `case` 后当前是表达式，不是模式匹配语法。
- 不支持 `case x if condition` guard。

## 8.3 暂不支持

以下能力当前不进入语言语法：

```jasper
case x if x > 0 => { }
case (x: int32, y: int32) => { }
```

未来如果重新设计模式匹配，应作为独立语言设计议题处理，而不是复用当前 `switch`。
