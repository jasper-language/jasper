# 第七章：类、接口、枚举与注解类型

## 7.1 类声明

```jasper
class Point {
    public var x: int;
    public var y: int;

    public constructor(x: int, y: int) {
        self.x = x;
        self.y = y;
    }

    public function distanceTo(other: Point): float {
        var dx = self.x - other.x;
        var dy = self.y - other.y;
        return sqrt(dx * dx + dy * dy);
    }
}
```

对应的 parser 规则：

```
normalClassDeclaration
    : modifier* Class name=Identifier typeParameters?
      superclass? superinterfaces? classBody
    ;
```

### 继承与接口实现

```jasper
class Dog extends Animal implements Pet, Runnable {
    // ...
}
```

```
superclass → Extends typePostfix

superinterfaces → Implements typePostfix (',' typePostfix)* ','?
```

### 类体成员

```
classBody → '{' classBodyDeclaration* '}'

classBodyDeclaration
    : fieldDeclaration                    // var/const 字段
    | propertyDeclaration                 // @Getter/@Setter property
    | constructorDeclaration              // 构造器
    | methodDeclaration                   // 方法
    | normalClassDeclaration              // 内部类
    | enumDeclaration                     // 内部枚举
    | normalInterfaceDeclaration          // 内部接口
    | annotationTypeDeclaration           // 内部注解类型
    | ';'                                 // 空语句
    ;
```

```jasper
class Outer {
    // 实例字段
    public var count: int = 0;

    // 静态字段（类级别，所有实例共享）
    public static var totalCount: int = 0;

    // 方法
    public function add(x: int): void {
        self.count = self.count + x;
        Outer.totalCount = Outer.totalCount + 1;
    }

    // 构造器
    public constructor() { }

    // 内部类
    private class Inner {
        public function doSomething(): void { }
    }

    // 内部枚举
    public enum State {
        Active,
        Inactive,
    }
}
```

## 7.2 接口声明

接口定义契约，包含方法签名、常量要求和 property 约束：

```jasper
interface Drawable {
    public const MIN_SIZE: int;           // 要求实现类提供此常量

    public function draw(): void;         // 抽象方法

    public var area: float { get; }       // 要求实现 getter
    public var scale: int { get; set; }   // 要求同时实现 getter 和 setter
}
```

接口支持泛型和多继承：

```jasper
interface Comparable<T> {
    public function compareTo(other: T): int;
}

interface Serializable extends Comparable<Serializable> {
    public function serialize(): bytes;
}
```

对应的 parser 规则：

```
normalInterfaceDeclaration
    : modifier* Interface name=Identifier typeParameters? extendsInterfaces? interfaceBody
    ;

extendsInterfaces → Extends typePostfix (',' typePostfix)* ','?
```

接口体成员集合：

```
interfaceMemberDeclaration
    : constantDeclaration               // 常量要求
    | interfacePropertyDeclaration      // Property 要求（get/set）
    | methodDeclaration                 // 带默认实现的方法
    | interfaceMethodDeclaration        // 抽象方法
    | normalClassDeclaration            // 内部类
    | enumDeclaration                   // 内部枚举
    | normalInterfaceDeclaration        // 内部接口
    | annotationTypeDeclaration         // 内部注解类型
    | ';'                               // 空语句
    ;
```

## 7.3 枚举声明

```jasper
enum Color {
    Red,
    Green,
    Blue,
}
```

枚举常量可以带参数和类体：

```jasper
enum HttpStatus {
    OK(200),
    NotFound(404),
    InternalError(500);

    private var code: int;

    public constructor(code: int) {
        self.code = code;
    }
}
```

枚举可以实现接口：

```jasper
enum Operation implements Describable {
    Add { public function describe(): string { return "addition"; } },
    Subtract { public function describe(): string { return "subtraction"; } },
    Multiply { public function describe(): string { return "multiplication"; } },
    ;
}
```

对应的 parser 规则：

```
enumDeclaration
    : modifier* Enum name=Identifier superinterfaces? enumBody
    ;

enumBody
    : '{' (enumConstant (',' enumConstant)* ','?)? ','? enumBodyDeclarations? '}'
    ;

enumConstant
    : modifier* Identifier arguments? classBody?
    ;

enumBodyDeclarations
    : ';' classBodyDeclaration*
    ;
```

## 7.4 注解类型声明

注解类型使用 `@interface` 语法声明：

```jasper
@interface Author {
    public string name();
    public int version() default 1;
}

@interface SuppressWarnings {
    public string[] value();
}
```

使用注解：

```jasper
@Author(name = "Alice", version = 2)
class MyClass { }

@SuppressWarnings({"unchecked", "deprecation"})
function legacyCode(): void { }
```

对应的 parser 规则：

```
annotationTypeDeclaration
    : modifier* '@' Interface name=Identifier annotationTypeBody
    ;

annotationTypeBody
    : '{' annotationTypeMemberDeclaration* '}'
    ;

annotationTypeMemberDeclaration
    : annotationTypeElementDeclaration     // type name() default?;
    | constantDeclaration                  // 常量
    | (normalClassDeclaration | enumDeclaration)
    | (normalInterfaceDeclaration | annotationTypeDeclaration)
    | ';'
    ;

annotationTypeElementDeclaration
    : modifier* typeOrPrimitive name=Identifier '(' ')' ('[' ']')* defaultValue? ';'
    ;
```

## 语法对照

| 特性 | Parser 规则 | 对应代码 |
|------|------------|---------|
| 类 | `normalClassDeclaration` | `class A extends B implements C { }` |
| 继承 | `superclass` | `extends BaseClass` |
| 接口实现 | `superinterfaces` | `implements I1, I2` |
| 类体 | `classBody` | `{ field; method(); constructor() }` |
| 实例字段 | `fieldDeclaration` | `public var x: int;` |
| 静态字段 | `fieldDeclaration`（加 `static` 修饰符） | `public static var x: int;` |
| 接口 | `normalInterfaceDeclaration` | `interface I extends J { }` |
| 接口成员 | `interfaceMemberDeclaration` | 方法/常量/property/内部类型 |
| 枚举 | `enumDeclaration` | `enum E { A, B, C }` |
| 枚举常量 | `enumConstant` | `A(args) { classBody }` |
| 注解类型 | `annotationTypeDeclaration` | `@interface A { }` |
| 注解元素 | `annotationTypeElementDeclaration` | `string name() default "";` |
